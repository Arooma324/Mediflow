#Mediflow
MediFlow is a smart Hospital Queue and Appointment Management System developed to improve the efficiency of patient management in small and medium-sized hospitals and clinics. The project replaces traditional paper-based processes with a digital solution that simplifies patient registration, appointment scheduling, queue management, and doctor coordination. Built using Python, Flask, and SQLite following the Model-View-Controller (MVC) architecture, the system is lightweight, easy to deploy, and capable of running on a local machine without requiring an internet connection. This makes it especially suitable for healthcare facilities operating in areas with limited connectivity.

The system provides three role-based interfaces for administrators, doctors, and receptionists. Administrators can manage users, monitor hospital activities, generate reports, and control doctor availability. Receptionists are responsible for registering patients, booking, rescheduling, or canceling appointments, and managing the live patient queue. Doctors can view their appointment schedules, monitor waiting patients, and track the consultation status of each patient.

MediFlow incorporates three important software design patterns to improve flexibility and maintainability. The Observer Pattern ensures that all relevant users receive real-time updates whenever queue information changes. The State Pattern manages the different stages of a patient's consultation process, allowing smooth transitions between waiting, in consultation, and completed states. The Command Pattern enables Undo and Redo functionality for appointment-related actions, reducing the impact of accidental scheduling errors and improving operational reliability.

The system stores all information securely in a local SQLite database, including patient records, doctor information, user accounts, appointments, and queue details. Its modular MVC architecture separates business logic, user interface, and data management, making the application easier to maintain, test, and expand with future features.

Overall, MediFlow offers a practical, affordable, and user-friendly healthcare management solution that enhances hospital workflow, minimizes manual errors, reduces patient waiting times, and improves communication between hospital staff. By combining modern software engineering principles with real-world healthcare requirements, the project demonstrates how technology can streamline hospital operations while remaining accessible to organizations with limited technical and financial resources.
